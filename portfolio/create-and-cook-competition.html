Project Summary

Client / Project

  • Client: TOOKE
  • Project: Create & Cook Competition
  • Scope: Event branding + print collateral
  • Location: Kampala - Uganda
  • Timeline: Concept → proofing → production support

What We Delivered

  • Event creative direction + key visuals
  • Poster and social promo layouts
  • Banners / backdrops (print-ready)
  • Final export packs (web + print)

Tools and Materials Used

  • Tools: Adobe Photoshop, Illustrator, InDesign
  • Workflow: mockups, approvals, prepress checks
  • Materials: banners, posters, sticker/label stock (as needed), laminated finishing

Goal, Scope & Result

Goal

Build a vibrant event identity that attracts participants, communicates key information clearly, and looks consistent across digital and print touchpoints.

What We Did

Developed a bold visual direction, designed event collateral, prepared print-ready files with correct sizing and bleed, and ensured consistency across formats.

Result

A cohesive campaign set that improved visibility, supported event communication, and delivered polished, on-brand materials ready for rollout.

Process

1) Concept & Direction

Defined the event tone, color direction, and layout style to match TOOKE’s brand and competition energy.

2) Design & Proofing

Created multiple collateral drafts and refined typography, hierarchy, and spacing for clarity at a distance.

3) Print Prep

Exported production-ready files with correct dimensions, bleed, safe zones, and format versions for different suppliers.

4) Rollout Support

Assisted with final checks for banners/posters and ensured visual consistency across digital promos and on-ground materials.
